Available Platforms

Windows

Download: ClipSync Windows (.exe)

Installation:

  1. Download the .exe file from the link above
  2. Run the installer as administrator
  3. Follow the installation wizard
  4. Launch ClipSync from Start Menu or Desktop

Linux

Download: ClipSync Linux (.deb)

Installation:

  1. Download the .deb file from the link above
  2. Open terminal in the download directory
  3. Run: sudo dpkg -i clipsync.deb
  4. If dependencies are missing, run: sudo apt-get install -f
  5. Launch ClipSync from applications menu

Uninstall (Linux):

To uninstall ClipSync on Linux, run: sudo apt remove clipsync
